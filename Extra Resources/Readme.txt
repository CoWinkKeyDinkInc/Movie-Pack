These files are other resources that you may add and remove into the resource pack for other effects.

- foilagecolor.png and grasscolor.png - Gives trees and grass an autumn-like colour.
Install on folder minecraft/textures/misc.

- paintings_kristoffer_zetterstrand.png - Paintings may be edited, use photo-editing software to change the templates.
Install on folder minecraft/textures/colormap.

- ascii.png - Changes the fonts to a higher resolution and more rounder font.
Install on folder minecraft/font.

- explosion.png - A filled coloured explosion.
Install on minecraft/textures/entity.

- greenscreen.png - Rename it to any block you like, and install it on minecraft/textures/
